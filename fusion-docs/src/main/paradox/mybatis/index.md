# MyBatis

`fusion-mybatis` 提供了对 MyBatis `SqlSessionFactory` 的管理功能，它依赖 @ref:[`fusion-jdbc`](../jdbc/index.md) 组件。

`fusion-mybatis` 集成了 [Mybatis-plus](https://mp.baomidou.com/)，对于刚从Java，特别是Spring过来的朋友来说可以更快速的使用Akka来实现自己的业务。
